[![Anaconda Platforms](https://anaconda.org/hep-forge/hathor/badges/platforms.svg)](https://anaconda.org/hep-forge/hathor)

Feedstock for [hathor](https://www.physik.hu-berlin.de/de/pep/tools/hathor.html) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

HAdronic Top and Heavy quarks crOss section calculatoR.

## Install

```bash
conda install -c hep-forge -c conda-forge hathor
```
